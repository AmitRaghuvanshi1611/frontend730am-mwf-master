Elements Selector

the Elements Selctor matches Elements by their tag name

Example :

        div
        {
            background-color : lightblue ;

        }   
        p
        {
            color : darkblue;
            font-size:18px;
            margin: 10px;
            padding:5px;
            font-family:'arial',sans-serif;
            line-height: 1.6;
        }
