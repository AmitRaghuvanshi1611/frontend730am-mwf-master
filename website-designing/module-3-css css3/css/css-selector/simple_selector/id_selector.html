Id Selector

The Id Selector matches by their unique id attiribute.

id is denoted by hash symbol (#) Followed by the ID name .

for example ,to select an eleaments with the id "header", you would use :

            #header 
            {
                background-color= white;
                padding : 10px
                width : 50%;
                font-size :24px;
                color : blue;
            }

in this exmple , the eleament with the id "header" will have a white background-color ,blue text , width 50% and specific padding .

and also id is unique so it is not repeated use

Example HTML:

            <div id ="header">This is the header</div>
            <p> This is paragraph is styled by the ID Selector.</p>
