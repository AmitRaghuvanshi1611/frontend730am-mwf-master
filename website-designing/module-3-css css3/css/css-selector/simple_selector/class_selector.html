Class Selector

The Class selector matches eleaments by their class attiribute.

its is denoted by a period (.) Followed by the class name.

For Example, to Select eleaments with the class "highlight", you would use:

class will be repeated more than one times in a section

class is not unique

            {

                padding : 0;
                margin : 0;

                color: white;
